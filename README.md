## Keen Keeper

## 📝 Description

Friend Tracker is a simple web application that helps users keep track of their interactions with friends. It allows users to log calls, texts, and video chats, and stay connected by maintaining regular communication.

---

## 🛠️ Technologies Used

* React.js
* React Router DOM
* Tailwind CSS
* Recharts
* React Icons
* React Toastify

---

## ⭐ Key Features

1. Track and log friend interactions (Call, Text, Video)
2. View interaction history through a timeline
3. Analyze communication patterns with a visual chart
